# Web3 LangChain Agent

## Overview

demo poc for langchain in web3

## Features

- Natural language processing
- Integration with price trends service
- TypeScript-based implementation
- Built on LangChain framework

## Usage Examples

```
npm install
```

test

```
npm run dev
```

compile

```

npm run build

```

chat terminal

```
npm run start
```

workchain

```
npm run start:workflow
```
